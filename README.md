## 🚀 Features
- **Form Validation:** ตรวจสอบข้อมูล ชื่อ, อายุ และอาชีพ ก่อนบันทึก
- **State Management:** ใช้ Stateful Widget จัดการข้อมูลในรายการ
- **Custom UI:** ออกแบบหน้าจอด้วย ListView และ Material Design
- **Routing:** ระบบเปลี่ยนหน้าไปมาระหว่างหน้าฟอร์มและหน้าแสดงผล

## 🛠 Tech Stack
- Flutter 3.x / Dart
- Google Fonts (Sarabun/Mitr)
